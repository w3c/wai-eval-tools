Acknowledgements for Web Accessibility Evaluation Tools List

Back to the Evaluations Tools List

The List of Web Accessibility Evaluation Tools was initially developed in 2001 by Wendy Chisholm and Judy Brewer. Since 2003, it is being managed by Shadi Abou-Zahra. The user interface is developed through the Education and Outreach Working Group (EOWG). The content was managed with help of the Evaluation and Repair Tools Working Group (ERT WG) until November 2015. Currently the content is being managed with the help of José Ramón Hilera González. We are thankful to the many contributors over so many years!
